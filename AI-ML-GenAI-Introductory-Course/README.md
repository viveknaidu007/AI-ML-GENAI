## 🌟 Overview

This comprehensive curriculum bridges **foundational data science** with **cutting-edge generative AI** and **autonomous agentic systems**. Designed for engineering students and professionals, it provides a structured pathway from Python fundamentals through advanced LLM deployment, multi-agent orchestration, and production MLOps.

### 🔄 Curriculum Evolution
```
Phase 1: Foundations (Modules 1-8)
   ↓
Phase 2: Generative AI Specialization (Gen AI Modules 1-16)
   ↓
Phase 3: Agentic AI Extension (Modules A1-A8)
   ↓
Phase 4: Advanced Production Topics (Modules M1-M15)
```

### ✨ Key Differentiators
- **End-to-End Coverage**: From data manipulation to autonomous agent deployment
- **Production-Ready Focus**: Emphasis on MLOps, security, scalability, and observability
- **Hands-On Projects**: Every module includes practical labs and real-world applications
- **Industry-Aligned**: Tools and frameworks used by leading AI teams (LangChain, LangGraph, MLflow, Feast, etc.)
- **Ethics & Safety**: Integrated responsible AI principles throughout

## 🎯 Learning Objectives

Upon completion, students will be able to:

### Core Competencies
✅ Master Python programming, data structures, and OOP for data science workflows  
✅ Perform advanced statistical analysis, hypothesis testing, and inferential reasoning  
✅ Design, train, and evaluate supervised/unsupervised ML models with proper validation  
✅ Build deep learning architectures (CNNs, RNNs, Transformers) for vision and NLP tasks  
✅ Apply prompt engineering, fine-tuning, and evaluation techniques for LLMs  

### Generative AI Specialization
✅ Implement retrieval-augmented generation (RAG) systems with hybrid retrieval strategies  
✅ Develop LangChain/LangGraph applications with memory, tools, and agent orchestration  
✅ Deploy generative AI solutions on AWS/Azure/GCP with cost optimization and monitoring  
✅ Evaluate LLM outputs using automated metrics, human preference modeling, and bias detection  

### Agentic AI Mastery
✅ Design autonomous agents with planning, memory, tool use, and self-reflection capabilities  
✅ Build multi-agent systems with role specialization, communication protocols, and consensus mechanisms  
✅ Implement production guardrails: input validation, output filtering, permission layers, and HITL workflows  
✅ Orchestrate stateful agent workflows with checkpointing, retry logic, and graceful degradation  

### Production Engineering
✅ Architect MLOps pipelines with experiment tracking, feature stores, and continuous training  
✅ Monitor production systems for drift, performance decay, and cost anomalies with SLOs  
✅ Scale AI services horizontally with caching, model routing, and circuit breakers  
✅ Ensure compliance with EU AI Act, GDPR, and NIST AI RMF through audit-ready design

## 📦 Prerequisites

### Technical Requirements
```yaml
Programming:
  - Python 3.8+ proficiency (functions, classes, file I/O)
  - Basic command-line and Git usage

Mathematics:
  - Linear algebra (vectors, matrices, eigenvalues)
  - Probability & statistics (distributions, hypothesis testing)
  - Calculus fundamentals (gradients, optimization)

Systems:
  - Familiarity with REST APIs and JSON
  - Basic cloud concepts (IaaS/PaaS)
  - Docker basics recommended

Hardware:
  - Minimum: 8GB RAM, 50GB storage, modern CPU
  - Recommended: 16GB+ RAM, NVIDIA GPU with CUDA, 100GB+ storage
```

## 🧪 Capstone Project: AI-Powered Autonomous Agent System

### Project Description
Design, build, and deploy an end-to-end agentic AI application that demonstrates mastery across the curriculum. Students choose from predefined tracks or propose original ideas.

### Sample Project Tracks
```
🔹 Track A: Autonomous Research Assistant
   - Multi-agent team (researcher, writer, editor, fact-checker)
   - RAG over academic papers + web search
   - Self-reflection and citation verification
   - HITL approval for publication

🔹 Track B: Business Intelligence Agent
   - Natural language to SQL + Power BI visualization
   - Anomaly detection + root cause analysis
   - Multi-turn conversational interface
   - Cost-optimized model routing

🔹 Track C: Creative Content Studio
   - Text-to-image generation with style control
   - Brand voice adaptation via fine-tuning
   - A/B testing of outputs with human preference modeling
   - Copyright compliance and attribution tracking

🔹 Track D: Custom Proposal
   - Student-defined problem with instructor approval
   - Must integrate ≥3 Agentic AI modules + ≥2 Advanced Topics
```

### Project Phases & Timeline
| Phase | Duration | Deliverables |
|-------|----------|-------------|
| **1. Ideation & Proposal** | Week 1-2 | Problem statement, literature review, architecture diagram, resource plan |
| **2. Data & Infrastructure** | Week 3-4 | Data pipeline, feature store setup, environment provisioning, EDA report |
| **3. Core Development** | Week 5-8 | Agent implementation, RAG integration, evaluation harness, safety controls |
| **4. Productionization** | Week 9-10 | Containerization, observability, CI/CD, load testing, cost profiling |
| **5. Documentation & Demo** | Week 11-12 | Technical docs, user guide, final presentation, peer review, open-source release |

### Evaluation Rubric
| Criterion | Weight | Description |
|-----------|--------|-------------|
| **Problem Definition & Scope** | 10% | Clarity, feasibility, alignment with curriculum concepts |
| **Data Quality & Pipeline** | 15% | Sourcing, preprocessing, versioning, governance |
| **Model/Agent Performance** | 25% | Task success, efficiency, robustness, evaluation rigor |
| **Code Quality & Architecture** | 20% | Modularity, documentation, testing, production patterns |
| **Safety, Ethics & Compliance** | 15% | Guardrails, auditability, bias mitigation, transparency |
| **Presentation & Communication** | 15% | Demo quality, documentation, stakeholder clarity |

## 📊 Assessment & Evaluation

### Continuous Assessment (100%)
```yaml
Weekly Assignments & Quizzes: 30%
  - Coding exercises, conceptual quizzes, peer reviews

Module Projects: 40%
  - Hands-on implementations with rubric-based grading

Capstone Project: 30%
  - End-to-end application with presentation and documentation
```

### Grading Scale
| Grade | Range | Description |
|-------|-------|-------------|
| **A** | 90-100% | Exceeds expectations; production-ready work |
| **B** | 80-89% | Meets expectations; solid understanding |
| **C** | 70-79% | Approaches expectations; needs refinement |
| **D/F** | <70% | Below expectations; significant gaps |

### Feedback Philosophy
- **Formative**: Weekly feedback on assignments to guide improvement
- **Summative**: Rubric-based evaluation with actionable comments
- **Peer Learning**: Structured code reviews and presentation feedback
- **Self-Assessment**: Reflection prompts to develop metacognitive skills

## 🚀 Getting Started

### Option 1: Full Curriculum (Self-Paced)
```bash
# 1. Clone the repository
git clone https://github.com/your-org/ai-curriculum.git
cd ai-curriculum

# 2. Set up environment
conda create -n ai-course python=3.11
conda activate ai-course
pip install -r requirements.txt

# 3. Start with Module 1
cd modules/01-python-fundamentals
jupyter lab
```

### Option 2: Modular Enrollment
```bash
# Install only what you need
# Example: Agentic AI specialization
pip install langchain langgraph openai pydantic redis qdrant-client

# Example: Production MLOps
pip install mlflow feast prefect bentoml evidently prometheus-client
```

### Option 3: Cloud Development Environment
```yaml
# Use pre-configured cloud workspace (example: GitHub Codespaces)
# .devcontainer/devcontainer.json includes:
{
  "image": "mcr.microsoft.com/devcontainers/python:3.11",
  "features": {
    "docker-in-docker": "latest",
    "aws-cli": "latest"
  },
  "postCreateCommand": "pip install -r requirements.txt"
}
```

### Quick Start Checklist
- [ ] Verify Python 3.10+ and pip installed
- [ ] Clone repository and install dependencies
- [ ] Configure cloud credentials (if using AWS/Azure/GCP)
- [ ] Set up vector database (local Qdrant or cloud Pinecone)
- [ ] Create LangSmith/Weights & Biases account for tracing
- [ ] Review Module 1 prerequisites and complete self-assessment quiz
